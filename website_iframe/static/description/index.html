Let users add iFrames to a client's Virtual Office.

If you use external tools like Google Analytics, Google Calendar, Google Data Studio you might want to give your users a simple way to embed these sites in their personal Odoo dashboard.


To add an iFrame go to Website Admin --> Iframe Dashboard --> iFrames and create a new entry with

  • Partner
  • URL
  • Active

Once created a new iFrame shows up on your Oficina Virtual of Odoo.
